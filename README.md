# NfbEcommerce

This is an E-commerce app that allows users to create account, authenticate, and add/subtract items to/from  shopping cart. 

## Live Link

[Heroku Live Site](https://newfullybakery.herokuapp.com)

## Technologies Used

- Angular
- Spring Boot
- PostgreSQL
- Okta
- Java
- JavaScript
- TypeScript
- Bootstrap
- Heroku hosting
